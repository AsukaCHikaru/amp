# AMP

AMP (stands for "Asuka's Markdown Parser") is a minimum markdown parser, written in TypeScript.

## Supported block types

- Heading
  - h1 ~ h6
- Styled texts
  - italic
  - bold
  - code
  - link
- Quote
- List
  - ordered (asterisks unsupported)
  - unordered
- Code
- Thematic break
